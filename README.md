# Glyphwind Samples

This repository contains sample demonstration projects built using the [Glyphwind Design System](https://github.com/brijeshcore/glyphwind).

Glyphwind is a monochromatic, highly geometric design system that emphasizes rapid, consistent UI development without the need to write custom CSS.

## Included Sample Projects

Inside this repository, you will find implementations highlighting the design system integrated with different frameworks and UI component libraries:

- **[Angular with Tailwind CSS](./angular-tailwind)**
  An example of how to configure and use Glyphwind's principles in an Angular environment.

- **[React with React Aria](./react-aria-sample)**
  A demonstration using React combined with [React Aria](https://react-spectrum.adobe.com/react-aria/), focusing on accessible UI components styled according to the Glyphwind system.

- **[React with Shadcn UI](./react-shadcn)**
  A showcase of Glyphwind applied to customized [Shadcn UI](https://ui.shadcn.com/) components in a React application.

---

For more details on the core design system, utility classes, and configuration, please check out the main [Glyphwind Repository](https://github.com/brijeshcore/glyphwind).
